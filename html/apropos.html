À propos du projet

Présentation

BeatScorer est un site web utilisent l’API ScoreSaber pour voir les informations des joueurs Beat Saber. ce site permet de consulter les profil, les scores récents, les maps, ainsi que de rechercher par filtrer.

Objectifs

Le but du projet est d’exploiter une API REST et de manipuler les données pour créer un site complet qui comprent :

  • La recherche de maps et de joueurs
  • Des filtres cumulables
  • Un système de pagination
  • Une gestion de favoris
  • Une page de détails des joueurs

API utilisée

Nom : ScoreSaber API

Documentation : docs.scoresaber.com

L’API fournit des données sur les joueurs, leurs scores et les leaderboards. Les réponses son au format JSON, et accessibles via des requêtes GET.

Endpoints exploités

  • /api/player/{id}/full — Profil complet d’un joueur
  • /api/player/{id}/scores — Scores récents
  • /api/leaderboards?search= — Recherche de maps

Déroulement du projet

Découpage

choix de l’API, tests des endpoints, création des pages HTML/CSS, développement des handlers Go, puis intégration des fonctionnalités (recherche, filtres, pagination, favoris).

Organisation

les tâches ont été organisé par priorité :les endpoints, les pages principales, les fonctionnalités obligatoires pour finir sur le design.

Gestion du temps

Les fonctionnalités obligatoires ont été faites en premier pour assurer le rendu du projet. API → pages → fonctionnalités → tests.

Documentation

la documentation officielle ScoreSaber se trouve sur swagger ce qui permet de tester les endpoints de puis le site
